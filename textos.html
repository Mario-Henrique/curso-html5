Trabalhando com textos, símbolos e emojis

A HTML é a espinha dorsal da web. Sua principal função é estruturar o texto, dando significado a cada palavra. Nesta página, vamos explorar as ferramentas que o HTML5 oferece para organizar, formatar e dar semântica ao seu conteúdo.

1. Elementos Estruturais Básicos

Elementos de Cabeçalho (Títulos)

As tags de cabeçalho (ou headings) são usadas para criar títulos e subtítulos. Elas são vitais para a organização do conteúdo e para o SEO (Search Engine Optimization).

A HTML usa 6 níveis de títulos, do mais importante ao menos importante, ou mais comum quando um subtítulo está abaixo de outro:

<h1> Título Principal da Página

Deve haver apenas um <h1> por página, representando o tema central.

<h2> Título de Seção

Usado para subdividir o conteúdo do <h1>.

<h3> Subtítulo de Seção

Usado para tópicos dentro do <h2>.

<h4> Título de Quarto Nível

<h5> Título de Quinto Nível
<h6> Título de Sexto Nível

Parágrafos e Quebras de Linha

<p> Parágrafo

A tag <p> é o elemento mais comum, usado para criar um bloco de texto. Os navegadores adicionam um pequeno espaço antes e depois de cada parágrafo automaticamente.

<br> Quebra de Linha

A tag <br> (de break) força uma
quebra de linha. É uma tag vazia (não precisa de fechamento) e deve ser usada apenas quando a quebra de linha é fundamental para o significado (como em poemas ou endereços).

<pre> Texto Pré-formatado

A tag <pre> é usada para exibir texto exatamente como ele foi escrito no código-fonte, mantendo espaços e quebras de linha. É ideal para exibir código ou arte ASCII:

    Exemplo:
        Isso não é
         um parágrafo.
    Os espaços são mantidos!

2. Elementos de Formatação e Ênfase

Ao trabalhar com texto, é fundamental entender que o HTML5 não se preocupa apenas com a aparência (que é papel do CSS), mas principalmente com o significado (semântica) do conteúdo. Usar a tag correta melhora a acessibilidade e a forma como robôs de busca e tecnologias assistivas (como leitores de tela) interpretam a página. Abaixo estão as tags de formatação de texto ativas, divididas por função.

Elementos Não Semânticos (Apenas Formatação Visual)

Estes elementos são usados principalmente para aplicar um estilo visual sem adicionar um significado estrutural ou de importância ao texto. No entanto, eles têm usos específicos e não estão depreciados, o que significa que ainda podem ser utilizados com o HTML5.

Elemento Exemplo Função e Uso Recomendado
<b> Texto em Destaque Usado para chamar a atenção do usuário, sem transmitir importância extra (ex: nomes de produtos ou palavras em um resumo). Se for importante, use <strong>.

Ex: Texto em destaque
<i> Voz Alternativa Usado para texto com uma voz ou humor diferente, como termos técnicos, pensamentos ou o nome de uma espécie. Se for ênfase tônica, use <em>.

Ex: Voz alternativa
<u> Texto Sublinhado Usado para sublinhar texto que possui uma anotação não textual, como uma marcação de erro ortográfico. É altamente desaconselhado para ênfase, pois pode confundir o usuário com um link.

Ex: Texto sublinhado
<span> Contêiner Genérico É um contêiner genérico em linha usado para agrupar pequenos trechos de texto, permitindo que o CSS ou JavaScript o estilize ou manipule. Não adiciona nenhuma semântica por si só.

Ex: Container genérico
<sub> H2O (Subscrito) Usado para exibir texto como subscrito, como em fórmulas químicas ou notações matemáticas.

Ex: H2O (Subscrito)
<sup> 22 (Sobrescrito) Usado para exibir texto como sobrescrito, como em expoentes ou notas de rodapé.

Ex: 22 (Sobrescrito)

3. Elementos de Citação e Código

Usados para indicar que o texto tem uma fonte externa ou que representa um tipo específico de conteúdo técnico.

Citações

A tag <blockquote> é usada para citações longas que ocupam um parágrafo inteiro ou mais, geralmente retiradas de outra fonte.

"O HTML5 trouxe um conjunto de tags semânticas que revolucionaram a estruturação da informação na Web, tornando-a mais acessível e compreensível." - Um Web Designer

A tag <q> é usada para citações curtas que ficam dentro de um parágrafo normal. Já a tag <cite> é usada para o título da obra de onde a citação foi tirada, como visto no exemplo do blockquote acima ao citar Um Web Designer como alguém que comentou sobre o HTML5.

Conteúdo Técnico

  • <code> Código: Usado para exibir trechos de código de programação.
  • <kbd> Entrada de Teclado: Representa uma entrada do usuário pelo teclado ou voz (ex: "Pressione Ctrl + C").
  • <samp> Saída de Programa: Representa uma saída ou resultado de um programa de computador.
  • <var> Variável: Indica uma variável em um contexto matemático ou de programação.

4. Símbolos Especiais e Emojis: O Poder do Unicode

Símbolos especiais e emojis são, na essência, caracteres. O HTML lida com eles de duas maneiras principais: através das Entidades de Caracteres e do Padrão Unicode, dependendo de como você deseja inseri-los.

Entidades de Caracteres (Símbolos HTML)

Entidades de Caracteres são códigos que você usa no HTML para exibir:

1) caracteres reservados que têm um significado especial no HTML (como os sinais de menor e maior que delimitam as tags); ou

2) caracteres que não estão disponíveis no teclado padrão (como frações ou moedas).

Elas são sempre iniciadas por um E comercial (&) e terminadas por um ponto e vírgula (;). Existem duas formas de usar a entidade: Nomeada (com um nome fácil de lembrar) ou Numérica (com o número do caractere).

Exemplos de Entidades Essenciais

Nome da Entidade Código Numérico Resultado
&lt; &#60; < (Sinal de Menor)
&gt; &#62; > (Sinal de Maior)
&amp; &#38; & (E Comercial)
&copy; &#169; © (Direitos Autorais)
&trade; &#8482; ™ (Marca Registrada)
&nbsp; &#160; (Espaço Não Quebrável)*
* O &nbsp; é crucial para inserir múltiplos espaços em branco, já que o HTML ignora espaços extras.

Emojis e Caracteres Unicode Complexos

Emojis não são mais do que caracteres de texto complexos definidos pelo padrão Unicode. Para que eles funcionem corretamente, é obrigatório que o cabeçalho do seu HTML contenha <meta charset="UTF-8">, pois o UTF-8 é o único conjunto de caracteres capaz de codificar a vasta gama de símbolos e emojis modernos.

Você pode inserir emojis de duas maneiras:

  1. Colando o Emoji Diretamente: É a forma mais fácil e mais comum hoje, desde que seu arquivo HTML esteja em UTF-8.
  2. Usando o Código Numérico: É o método mais confiável, pois usa o número do Unicode (geralmente em formato decimal ou hexadecimal) para chamar o caractere.

Exemplos de Emojis Usando Código Numérico

Geralmente usamos a notação hexadecimal do Unicode (&#x[HEX];) ou decimal (&#[DEC];) para garantir a exibição, embora colar o emoji diretamente seja o mais prático.

Código Numérico (Hexadecimal) Resultado
&#x1F600; 😀 (Rosto Sorrindo)
&#x1F44D; 👍 (Polegar para Cima)
&#x2764; ❤ (Coração)
&#x1F525; 🔥 (Fogo)
&#x1F31F; 🌟 (Estrela Brilhante)
