Formatos de imagem

Formato de imagem é o tipo de arquivo gráfico que o navegador é capaz de interpretar e exibir na página da web. Alguns formatos mais conhecidos são:

  • .jpg/.jpeg - Possui melhor compressão de dados, ou seja, é um tipo de arquivo menor comparado a outros tipos.
  • .png - É um pouco maior mas permite fundo transparente.
  • .gif - Possui transparência e animação porém pouca qualidade.
  • .svg - É uma imagem vetorial. Pode ser expandido para qualquer tamanho sem perder resolução.

Adicionando imagem com <img>

A tag <img> permite adicionar imagens em sua página. Com ela você usa dois atributos muito importantes:

  • src: significa source. Define o caminho onde está a imagem.
  • alt: É um texto alternativo utilizado por leitores de tela para acessibilidade por pessoas com deficiência visual.
<img src="caminho\arquivo.extensão" alt="Imagem de exemplo">

Adicionar ícones Favicon

Este é o ícone que aparece junto ao seu title na aba do navegador. Você pode adicionar esta linha em seu <head>.

Como o ícone é um recurso externo vamos usar a tag <link> com os atributos rel, href e type.

  • rel - Define a relação entre o documento atual e o recurso vinculado.
      Valores comuns:
    • stylesheet: Indica que o arquivo vinculado é uma folha de estilo, usado para aplicar estilos CSS a página.
    • icon: Especifica o ícone do site (favicon) que aparece na aba do navegador.
    • preload: Usado para carregar um recurso de forma prioritária, antes que o navegador precise dele.
    • prev ou next: Indicam a relação com a página anterior ou seguinte de uma série.
  • href: Especifica a URL (caminho) do recurso externo ao qual a página está sendo vinculada.
  • type: Especifica o tipo de mídia do recurso vinculado. Para folhas de estilo CSS, o valor é text/css.
  • media: Define o tipo de mídia ou dispositivo para o qual a folha de estilo é otimizada, permitindo aplicar estilos diferentes para telas, impressões, etc.
      Valores comuns:
    • screen: Para telas de computador.
    • print: Para impressão.
    • all: Aplica a todos os tipos de mídia.

Logo para adicionar o ícone usamos dessa forma:

<link rel="shortcut icon" href="icone.ico" type="image/x-icon">
