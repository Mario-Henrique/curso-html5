4. Listas

Listas são essenciais para organizar itens de forma clara.

Lista Não Ordenada (<ul>)

A tag <ul> (de *unordered list*) cria uma lista onde a ordem dos itens **não** é importante. Cada item é marcado com a tag <li>.

  • Ponto de café da manhã
  • Ponto de almoço
  • Ponto de jantar

Lista Ordenada (<ol>)

A tag <ol> (de *ordered list*) cria uma lista onde a **sequência** ou a ordem dos itens **é importante**. O navegador enumera os itens automaticamente.

  1. Passo 1: Abrir o editor de código.
  2. Passo 2: Criar o arquivo HTML.
  3. Passo 3: Salvar e visualizar no navegador.

Lista de Descrição (<dl>)

A tag <dl> (de *description list*) é usada para listar termos e suas definições. É ideal para glossários.

<dt> Termo:
Abreviação de *Description Term*.
<dd> Definição:
Abreviação de *Description Definition*.
